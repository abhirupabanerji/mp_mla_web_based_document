# Public Facility Registration Page
![register](/img/publicreg.png)
## Purpose
This feature allows representatives to register their public facilities without logging into the system. It ensures that only valid and verified facilities are added through KYC document submission and admin approval.
## How to Use
-	Go to the Public Registration page (available on the login page).
-	Fill in required details such as personal information, facility details, and head/in charge details. 
-	Upload necessary KYC documents (facility proof, identity proof, and address proof). 
-	Submit the registration request for verification. 
-	Wait for admin approval; once approved, the facility will be added and visible in the system.
