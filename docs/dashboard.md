# Dashboard Module
![dashboard](/img/dashboard.png)
### Purpose
The Dashboard provides a real-time overview of constituency activities and performance metrics.
### How to Use
-	Log into the system.
-	The dashboard opens automatically.
-	View key metrics such as:
    *	Active wards
    *	Open Requests
    *	Running projects
    *	Total Registered Voters
    *	Meetings Scheduled
-	Visual charts for data insights
