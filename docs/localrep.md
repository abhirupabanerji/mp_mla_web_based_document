# Local Representatives Module
![local rep](/img/localrep.png)
## Purpose
The Local Representatives Module maintains records of influential individuals within the constituency.
## How to Use
-	Open Local Representatives module.
-	Click Add Leader.
-	Enter details such as:
    -	Name
    -	Category
    -	DOB
    -	Designation
    -	Ward
-	Save the record.
-	Representatives can also be added in bulk using the available bulk upload feature (download the sample template and add rows accordingly).
-   View representative details and related insights, and use search and filters to find specific individuals.

Leader information supports outreach and engagement activities.
