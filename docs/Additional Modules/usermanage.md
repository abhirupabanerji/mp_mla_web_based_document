# User Management Module
![usermanage](/img/user.png)
## Purpose
The User Management module allows the System Administrator to create and manage user accounts within the system. 
It enables the administrator to assign roles, control user access, and manage permissions for different modules. 
## How to Use
-	Log in as System Administrator or MP/MLA.
-	From the Administration section in the left navigation menu, click User Management.
-	The system will display a list of all users with details such as Name, Role, Status, Last Login, and Created By.
-	To add a new user, click the Add User button on the top right, enter the required details (name, email, password, nd role), and save.
-	To search for a user, use the search bar and enter the user's name, email, or phone number.
-	Use the Role filter to view users based on their assigned roles and the Status filter to view users based on ccount status (Active/Inactive/Suspended).
-	To edit a user, click the Actions (three dots) next to the user and select the edit option to update user details r role permissions.
-	The administrator can also activate or deactivate users from the Actions menu to control whether a user can log into the system.
