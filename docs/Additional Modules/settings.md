# Settings Module
![settings](/img/systemsettings.png)
## Purpose
The System Settings module allows users to manage their account preferences and system configurations. 
## How to Use
-	Log in as System Administrator or MP/MLA.
-	From the main navigation menu, open Settings.
-	The System Settings page will display different configuration sections such as Profile, Language, Location, General, Branding, Security, Notifications, and Backup.
-	Select the required section from the left panel depending on the settings you want to manage.

Profile Settings
- View and update personal account information 

Language
- Allows users to change the display language of the system.

Location
- Enables users to set location preferences for the platform.

General
- Used to configure general system preferences related to organization or system locale.

Branding
- Allows administrators to customize the system appearance, including logo, theme colors, and branding elements.

Security
- Provides options to manage security settings such as passwords, active sessions, and two-factor authentication (2FA).

Notifications
- Allows configuration of notification preferences including SMS, WhatsApp, and email notifications.

Backup
- Enables administrators to manage scheduled backups and maintain system data securely.
