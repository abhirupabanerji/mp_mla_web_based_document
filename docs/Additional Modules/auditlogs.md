# Audit Logs Module
![auditlogs](/img/audit.png)
## Purpose
The Audit Logs module allows administrators to track and monitor all activities performed within the system. It records actions such as user logins, updates, notifications sent, and other system operations. This helps maintain transparency, improves security, and provides a record for reviewing system activity and troubleshooting issues.
## How to Use
-	Log in as System Administrator or MP/MLA.
-	From the navigation menu, open Audit Logs.
-	The Audit Logs dashboard will display a summary including:
    -	Total Logs 
    -	Today
    -	This Week 
    -	Active Users 
-	View activity insights such as Top Modules and Actions to understand which system areas are most frequently used.
-	Use the search bar to find specific logs by entering details such as description, user name, or IP address.
-	Apply filters to refine the log results
 
-	Review the activity records displayed in the table
-	Click the view icon in the details column to see additional information about a specific log entry.
-	If required, click Export All to download the audit logs for reporting or record-keeping purposes.
