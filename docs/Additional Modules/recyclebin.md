# Recycle Bin
![recyclebin](/img/recyclebin.png)
## Purpose
This module is used to view, restore, or permanently delete records that have been removed from different modules in the system. It helps in recovering accidentally deleted data and managing system cleanup.
## How to Use
-	Log in as System Administrator or MP/MLA.
-	Use the search bar to find specific deleted records. 
-	Filter records by module or entity type for easier navigation. 
-	Click on Restore to recover a deleted record. 
-	Use the delete option to permanently remove records from the system.
