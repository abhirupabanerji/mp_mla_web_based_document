# Community Groups Module
![comm groups](/img/communitygroups.png)
## Purpose
This module tracks community organizations within the constituency.
Examples include:
-	Resident Welfare Associations (RWA)
-	Clubs
-	Youth groups
-	Women's groups
-	Market associations
-	Sports teams
## How to Use
-	Open Community Groups module.
-	Click Add Group.
-	Enter group information and ward location.
-	Save the details.
-	Bulk Upload feature is available for adding multiple groups in one go.
-	Use the data for targeted outreach or communication. 
-   Browse group records, view summary charts, and use search and filters for easy access to required data.
