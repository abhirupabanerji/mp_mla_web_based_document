# Funds Module
![funds](/img/funds.png)
## Purpose
The Funds Module tracks constituency development funds such as MPLAD or MLALAD.
## How to Use
-	Open Funds module.
-	Add new fund allocations.
-	Record:
    *	Fund type
    *	Financial year
    *	Initial allocation
-	View available funds remaining.
-	Link funds to projects for tracking funds utilization for started projects.
