# Projects Module
![projects](/img/projects.png)
## Purpose
The Projects Module tracks development projects and budget utilization.
## How to Use
-	Navigate to Projects.
-   View all projects records along with visual charts and use the search bar to quickly find specific records.
-	Click New Project.
-	Enter project details:
    -	Project name
    -	Category
    -	Ward
    -	Department
    -	Contractor details
    -	Start / end date
    -	Budget allocation
- 	Bulk Upload feature is available, allowing users to add multiple projects using the provided sample template.
-	Update progress periodically.
-	Mark project as Completed when finished.
-	Use filters to filter projects by department / status / category.

Project updates automatically reflect on the dashboard and fund tracking.
