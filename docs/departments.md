# Departments Module
![departments](/img/departments.png)
## Purpose
This module manages government departments responsible for projects and grievance resolution.
All government departments active in the constituency are listed here — you can add new departments and assign them to relevant wards or projects for better coordination.
## How to Use
-	Open Departments module.
-	Add or view departments.
-   View department records, and utilize search for quick navigation.
-	Bulk Upload feature is available for adding multiple department records.
-	Assign Requests or projects to departments.
-	Track department performance through reports and dashboard metrics.
