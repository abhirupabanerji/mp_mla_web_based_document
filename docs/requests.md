# Public Requests Module
![requests](/img/publicrequests.png)
## Purpose
The Public Requests Module helps track citizen complaints and service requests.
## How to Use
-	Open the Public Requests module.
-	View all requests, analyse data through charts, and use filters and search options to track and manage specific cases.
-	Click New Request
-	Enter details such as:
    *	Subject
    *	Description
    *	Category
    *	Priority
    *	Source
    *	Assignment details
-	Save the record.
-	Bulk Upload feature is available to add multiple requests at once.
-	Update grievance status as:
    *	Open
    *	In Progress
    *	Resolved
-	Monitor grievance trends and resolution rates.-
