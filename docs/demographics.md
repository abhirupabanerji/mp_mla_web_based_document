# Wards and Demographics
## Wards Module
![wards](/img/wards.png)
### Purpose
The Wards Module manages geographic divisions within the constituency.
### How to Use
-	Open Wards module.
-   View ward details and charts, and use available filters and search functionality to find specific wards.
-	Demographics data will also be added from this module. (Age distribution / Social categories / Religion / Births and Deaths management / new eligible voters tracking etc.)
-	Bulk Upload feature is available to upload ward details in bulk.
-	All system modules are linked with wards including:
    *	Requests
    *	Projects
    *	Public Facilities
    *	Local Representatives 
    *	Community groups

Ward mapping allows ward-level performance analysis and reporting. 
## Demographics Module
![demographics](/img/demographics.png)
### Purpose
The Demographics Module stores and analyses population statistics of the constituency.
### How to Use
-	Open Demographics module.
-	View population data categorized by different demographic factors.
-   View demographic data through charts and apply filters to analyze specific segments.
-	Use the data for planning, analysis, and electoral strategy.
