# Users and Roles
## 2.1 MP / MLA (Owner / Decision Maker)

-   View the Dashboard for a complete constituency overview
-   Monitor Projects — progress, status, and timelines
-   Review Funds Utilization — track spending and remaining budget
-   Approve or follow up on Public Requests raised by citizens
-   View Demographics for planning and awareness
-   Access Reports for press, government submissions, or personal review
-   Approve institutions registered by heads on public registration page
-   Schedule and review Meetings & Events
-   View Birthday reminders for key constituents and representatives, also greet them when required
-   Oversee Community Groups and Local Representatives
-   Grant or review top-level Permissions for staff access

## 2.2 System Admin (Technical Controller)

-   Manage User Management — create, edit, deactivate user accounts
-   Configure Roles & Permissions — define what each role can see or do
-   Manage Settings — system configuration, preferences
-   Monitor Audit Logs — track all user activity and changes
-   Manage Recycle Bin — review and restore or permanently delete records
-   Add and manage Wards and their mappings across modules
-   Add and manage Departments in the system
-   Ensure data integrity across all 18 modules
-   Handle onboarding of new staff onto the platform

## 2.3 Personal Assistant / PA (Day-to-Day Operator)

- Add and update Projects and their progress
- Record and track Public Requests — log, assign, and follow up
- Update Public Facilities data ward-wise
- Manage Meetings & Events — schedule, update, record minutes
- Maintain Community Groups and Local Representatives records
- Update Demographics data as needed
- Add Birthday entries for constituents and representatives
- Generate and share Reports for the MP/MLA
- Coordinate with departments and update relevant records
- Act as the primary data entry and maintenance operator
