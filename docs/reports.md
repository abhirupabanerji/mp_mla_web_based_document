# Reports Module
![reports](/img/reports.png)
## Purpose
The Reports Module generates detailed reports for governance review and transparency.
## How to Use
-	Open Reports module.
-	Select the module whose detailed report you want to see:
    *	Public Requests reports
    *	Project reports
    *	Ward performance reports
    *	Public Facilities reports
    *	Demographics reports
    *	Monthly reports
-	Apply filters (Ward, month).
-	Generate report.
-	Export reports (XLSX) for meetings or audits.
