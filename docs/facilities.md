# Public Facilities Module
![facilities](/img/facilities.png)
## Purpose
The Public Facilities Module maintains a directory of Schools, Hospitals, Government offices, NGOs, Police Stations and other Public Facilities in the constituency.
## How to Use
-	Open Public Facilities module.
-	Click Add Institution.
-	Enter details such as:
    -	Institution name
    -	Category
    -	Ward
    -	Status
    -	Address
    -	In charge
    -	Contact details
-	Save the entry.
-	Bulk Upload feature is available, allowing users to add multiple facilities using the provided sample template.
-	Use filters to view Public Facilities ward-wise, category and status wise
-   View facility records and related charts, and apply search to locate specific entries.
