# Birthdays Module
![birthdays](/img/birthday.png)
## Purpose
The Birthdays Module tracks birthdays of Local Representatives and community members.
## How to Use
-	Open Birthdays module.
-	View upcoming birthdays.
-	Send greetings.
Birthday information is automatically pulled from the Local Representatives Module.
